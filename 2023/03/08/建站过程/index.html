GitHub Pages

到GitHub Pages网站看看，有个简单的使用流程

GitHub Pages | Websites for you and your projects, hosted directly from your GitHub repository. Just edit, push, and your changes are live.

step 1

在自己的GitHub上新建一个库，名字是 .github.io 这里的 username 就是自己账号的用户名

step 2

网页上是的操作

其实呢，在repository页面里直接建一个index.html,内容是Hello World! 是一样的效果

以上步骤之后，打开 https://username.github.io 正常的话是能看到Hello World! 的

这里只是算一个测试性质的吧，成型后的操作都交给Helo就行

Hexo

GitHub Pages是支持 Jekyll、Hugo、Hexo编译静态资源的

Jekyll需要 Ruby 开发环境，官方不建议在 Windows 下运行，但想要用也是有办法的，官方文档有写。
Hugo 是用 Go 语言写的，支持多个平台。
Hexo 需要安装 nodejs

我这里因为用过 nodejs 所以就选择了hexo ( Hugo 官网主题页面似乎没有 Hexo 多)

Hexo 中文官网 Hexo

安装

可以用npm直接安装

npm install hexo-cli -g

建站

之后打开 http://localhost:4000 接可以看到默认的一个介绍页面

配置

详细的配置参数可以在官网的文档里找到，可以在 _config.yml 中修改大部分的配置。
